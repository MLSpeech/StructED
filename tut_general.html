Tutorials

In order to use StructED to a specific task one must implement three interfaces which contains four functions:

  1. Loss function from TaskLoss interface.
  2. An argmax over the linear discriminant function from Prediction interface.
  3. An argmax over the loss-augmented linear discriminant function from Prediction interface.
  4. A function for computing Phi from PhiConverter interface.
